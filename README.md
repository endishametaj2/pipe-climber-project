# Pipe Climber Project

# Overview
Team-based mechanical engineering design project involving the CAD design and construction of an external pipe climber to meet IMechE challenge specifications.

# Objectives
- Design a mechanism capable of climbing an external pipe
- Meet challenge constraints and engineering specifications
- Develop practical CAD and prototyping skills

# My Contributions

- In charge of coming up with the electrical systems involved with the design
